## 🛠️ Local Development

   ```bash
   git clone https://github.com/adithya-retheep/competition-for-april-fool.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the dev server**:
   ```bash
   npm run dev
   ```
